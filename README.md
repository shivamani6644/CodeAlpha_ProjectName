# Hangman Game

## Description
A simple text-based Hangman Game built using Python. The player guesses a hidden word one letter at a time. The game allows a maximum of 6 incorrect guesses.

## Features
- Random word selection
- Input validation
- 6 incorrect attempts allowed
- Win and lose conditions
- Console-based game

## Technologies Used
- Python
- Random Module

## How to Run
1. Open terminal.
2. Navigate to project folder.
3. Run:

python hangman.py
